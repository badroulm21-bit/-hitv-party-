# HITV Party Application

## Overview
HITV Party is a social application designed to allow users to create and manage parties, inviting friends and keeping track of events.

## Features
- Create and organize parties
- Invite friends via social media or email
- RSVP functionality
- Event reminders
- Customizable party themes

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/badroulm21-bit/-hitv-party-.git
   ```
2. Change into the project directory:
   ```bash
   cd -hitv-party-
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
